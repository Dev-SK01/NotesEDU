## Installation

- Clone This repo to your machine....
```bash
git clone https://github.com/Dev-SK01/NotedEDU.git
```
- Move to the cloned Directory
```bash
cd notes-edu 
```
- Use npm to install all dependencies
```bash
 npm install 
```
- To Run 
```bash
npm start
```
